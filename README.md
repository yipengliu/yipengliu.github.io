# 刘翼鹏的个人学术网站

这是电子科技大学信息与通信工程学院教授刘翼鹏的个人学术网站，使用 Jekyll 和 GitHub Pages 构建。

## 本地运行

如果需要在本地预览网站，请按以下步骤操作：

1. 安装 Ruby 开发环境
```bash
sudo apt install ruby-dev ruby-bundler nodejs
```

2. 安装依赖
```bash
bundle install
```

3. 启动本地服务器
```bash
bundle exec jekyll serve
```

网站将在 `http://localhost:4000` 运行。
